# Rental App – Smart Property Management Solution

Rental App is a full-stack application built to streamline the rental experience for landlords and tenants, mainly focusing on the Zanzibar Islands. It offers property listing management, intelligent search functionality, secure user authentication, and Sheha authorization and a responsive interface designed for ease of use across devices.

This project demonstrates practical implementation of modern web development concepts, database integration, and user experience design while solving real-world property rental challenges.

Features:
-User registration and authentication
-Sheha verification on house status
-Property listing and management
-Advanced property search and filtering
-Property details with images and descriptions
-Booking and inquiry system
-Responsive and user-friendly interface
-Secure data management
